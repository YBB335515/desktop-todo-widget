# context-check Skill

## 触发时机
每次对话任务完成时（或用户输入 `/context-check`），显示会话资源状态。

## 行为流程

### 1. 显示信息
向用户展示以下三项信息：

```
📁 当前目录:   {当前工作目录}
🧠 模型:       {当前使用的模型名称}
📊 上下文:     {已用百分比}% 已用 | 剩余约 {剩余百分比}% 可用
```

### 2. 阈值提醒
- **上下文剩余 ≥ 60%** → 仅显示信息，无需特别提示
- **上下文剩余 < 60%** → 显示后追加提醒，建议用户执行 `/compact` 压缩上下文

提醒文案：
> ⚠️ 上下文剩余不足 60%，建议输入 `/compact` 压缩对话历史以保持响应质量。

### 3. 获取上下文信息
- 目录：从当前 session 工作目录获取
- 模型：从环境信息中的 model 字段获取
- 上下文百分比：从系统提示中的 context usage 信息获取（如 `65% used`），计算剩余百分比 = 100 - 已用百分比
